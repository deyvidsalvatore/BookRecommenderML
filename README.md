# Book Recommender

Práticas usando o machine learning para recomendar livros semelhantes.

* Pandas
* Numpy
* Matplotlib
* Seaborn
* Scikit-Learn

### Instalação com Dockerfile
```
docker build -t book-recommender-ml .
docker run -p 8501:8501 book-recommender-ml
```
